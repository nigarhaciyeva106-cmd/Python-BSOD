# 💀 Python BSOD Prank Simulator

Bu layihə Python dilində yazılmış, əyləncə və zarafat məqsədli bir simulyatordur. 

## 🚀 Xüsusiyyətlər
- **Mavi Ekran (BSOD):** Oyunun sonunda saxta və zərərsiz bir mavi ekran çıxır.
- **Alt+F4 Bloklama:** Proqramın bağlanmasının qarşısını alır.
- **Topmost:** Həmişə digər pəncərələrin üzərində qalır.

## ⚠️ Necə Bağlamalı?
Bu sadəcə bir zarafatdır! Proqramı bağlamaq üçün klaviaturada **Ctrl + Shift + Esc** düymələrini sıxaraq Task Manager-dən (Tapşırıq Meneceri) proqramı sonlandırın.
